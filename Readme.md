<h2 align="center"> 🕹️ Comandos para inicializar o projeto: <br>
<img src="https://img.shields.io/badge/Terminal_Integrado-000000?style=for-the-badge&logo=gnumetalinux&logoColor=white" height="25"></h2>

### Siga os passos abaixo para configurar o ambiente:

Após baixar o Github desktop: <br>
<img src="https://img.shields.io/badge/-GitHub%20Desktop-111827?style=flat-square&logo=github&logoColor=6F42C1" height="22"/>

```bash
# 1. Clone o repositório:
git clone [https://github.com/wqiluc/Cinema-Python-Flask](https://github.com/wqiluc/Cinema-Python-Flask)

# 2. Acesse a pasta do projeto:
cd Cinema-Python-Flask

# 3. Crie um ambiente virtual (recomendado):
python -m venv venv
source venv/bin/activate  # No Windows: venv\Scripts\activate

# 4. Instale as dependências:
pip install -r requirements.txt

# 5. Inicie a aplicação:
ls
cd BACKEND
ls
flask run
```
